### Selfbot and public bot

### For Termux
```bash
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
> git clone https://github.com/ground67/SelfWA
> cd SelfWA
> npm i
```

### Run
```
> npm start
```
